# سجل التغييرات - Lawyer Office Management System

## [1.0.0] - 2025-01-XX

### 🆕 ميزات جديدة

#### نظام المصادقة
- نظام تسجيل دخول برمز 6 أرقام (الرمز الافتراضي: 123456)
- إمكانية تغيير الرمز من إعدادات التطبيق
- حماية جميع المسارات بالتحقق من المصادقة

#### الهيئات القضائية (تم إعادة تصميمها بالكامل)
- **5 أقسام منظمة:**
  1. **القضاء العادي**: ولاية → مجلس قضائي → محكمة → قسم/غرفة
  2. **المحكمة العليا**: عرض الغرف بدون ارتباط بولاية
  3. **القضاء الإداري**: ولاية → محكمة إدارية استئنافية → محكمة إدارية
  4. **مجلس الدولة**: كيان مستقل
  5. **المحكمة التجارية المتخصصة**: ولاية → محكمة تجارية
- نموذج إضافة ديناميكي يتغير حسب نوع التنظيم القضائي
- عرض الهيئات الموجودة فقط (إخفاء الولايات الفارغة)
- إمكانية إضافة غرف/أقسام لكل هيئة قضائية

#### الموكلين
- إضافة/تعديل/حذف الموكلين
- الحقول: الاسم الكامل، الهاتف، العنوان، ملاحظات
- جميع الحقول اختيارية

#### القضايا (تم تحسينها بالكامل)
- أنواع القضايا: دعوى جديدة، استئناف، معارضة، طعن بالنقض
- حقول ديناميكية حسب نوع القضية:
  - **استئناف**: رقم القضية الابتدائية، المحكمة الأصلية
  - **معارضة**: رقم الحكم الغيابي، تاريخه، الجهة المصدرة
- **نظام الأطراف الجديد:**
  - **في حق (موكل)**: اختيار من قائمة الموكلين + الصفة
  - **ضد (خصم)**: الاسم، اللقب، الهاتف، العنوان، الصفة، المحامي (اختياري)
- حقول: رقم القضية، الولاية، الهيئة القضائية، الغرفة، الحالة، الأتعاب، الموضوع، ملاحظات

#### المحامين (جديد)
- إضافة/تعديل/حذف المحامين
- الحقول: الاسم، اللقب، الهاتف، العنوان المهني، المنظمة التابع لها
- ربط المحامي بنقابة المحامين

#### المنظمات (جديد)
- أنواع: نقابة محامين، مجلس قضائي، محكمة، أخرى
- الحقول: الاسم، النوع، العنوان، الهاتف، الولاية
- تصفية حسب النوع

#### الجلسات
- ربط الجلسات بالقضايا
- الحقول: التاريخ، سبب التأجيل، القرار، نص الحكم، ملاحظات

#### الرزمانة
- عرض شهري/أسبوعي/يومي
- أنواع الأحداث: جلسة، موعد، اجتماع، مهمة
- ربط الأحداث بالقضايا والجلسات

#### النسخ الاحتياطي
- تصدير كامل البيانات بصيغة JSON
- استيراد مع فحص سلامة البيانات
- التحقق من صحة الملف قبل الاستيراد

#### البحث الشامل (جديد)
- اختصار Ctrl+K للبحث السريع
- البحث في: القضايا، الموكلين، الجلسات
- نتائج فورية مع إمكانية التنقل المباشر

#### PWA - Progressive Web App
- دعم العمل بدون اتصال
- إمكانية تثبيت التطبيق على الجهاز
- أيقونات مخصصة (192x192, 512x512)
- ملف manifest.json

### 🔧 تحسينات تقنية

#### قاعدة البيانات
- الانتقال من Prisma إلى Drizzle ORM
- استخدام SQLite لتخزين البيانات محلياً
- جداول جديدة:
  - `organizations` - المنظمات
  - `lawyers` - المحامين
- تحديث جدول `case_clients`:
  - حقول الخصم: `opponent_first_name`, `opponent_last_name`, `opponent_phone`, `opponent_address`
  - حقل المحامي: `lawyer_id`
- نظام ترحيل تلقائي للأعمدة الجديدة

#### API Endpoints
- `GET/POST/PUT/DELETE /api/judicial-bodies` - الهيئات القضائية مع تنظيم حسب النوع
- `GET/POST/PUT/DELETE /api/lawyers` - المحامين
- `GET/POST/PUT/DELETE /api/organizations` - المنظمات
- `GET/POST/PUT/DELETE /api/cases` - القضايا مع الأطراف
- `GET/POST/PUT/DELETE /api/clients` - الموكلين
- `GET/POST/PUT/DELETE /api/sessions` - الجلسات
- `GET/POST/PUT/DELETE /api/calendar` - التقويم
- `POST /api/backup/export` - تصدير البيانات
- `POST /api/backup/import` - استيراد البيانات
- `GET /api/search` - البحث الشامل
- `GET /api/stats` - الإحصائيات

#### واجهة المستخدم
- تصميم RTL كامل للغة العربية
- قائمة جانبية قابلة للطي على الموبايل
- نظام ألوان موحد لكل نوع هيئة قضائية
- أزرار إضافة ديناميكية في النماذج
- تحسين رسائل الخطأ والنجاح
